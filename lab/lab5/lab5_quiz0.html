(Quiz) Some polymorphism puzzles

Consider the following class definitions. 
public class Superclass { 
 
  public void print ( ) { 
    System.out.println ("super"); 
  } 
}

public class Subclass extends Superclass { 
 
  public void print ( ) { 
    System.out.println ("sub"); 
  } 
}

Consider now the following program segments. For each, indicate whether it will produce (a) a compiler error message, (b) a runtime error message, or (c) one of the words "super" or "sub".
Program Segment Result 
Superclass obj1 = new Subclass ( ); 
obj1.print ( );
sub 
Subclass obj2 = new Superclass ( ); 
obj2.print ( );
compile-time error 
Superclass obj3 = new Superclass ( ); 
((Subclass) obj3).print ( );
runtime error 
Subclass obj4 = new Subclass ( ); 
((Superclass) obj4).print ( );
sub
